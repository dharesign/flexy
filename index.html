Flexy
a flexbox-based grid system
GitHub Project

What is flexy?

Flexy is a flexbox-based grid system. I wrote it after failing to find an existing solution which exactly matched my requirements. I wanted something very minimal (just a grid system), with no vendor-specific prefix as I deem them no longer necessary. I also wanted sizes to be specified in rem units.

How do I use it?

Flexy is available on GitHub. The master branch contains a single flexy.css file which you should download and include in your project as usual.

Below is a collection of examples of all of the different classes available.

Structure

Flexy is used by specifying the relevant classes. All classes are prefixed by flexy-.

Flexy splits the grid into six columns.

Flexy defines four responsive breakpoints, with the relevant classes being:

  • *-xs-*: Always applies
  • *-sm-*: Applies above 48rem
  • *-md-*: Applies above 62rem
  • *-lg-*: Applies above 76rem

Below a given breakpoint, columns defined with the relevant class degrade to be stacked vertically taking up the entire width available.

Auto Columns, Packed Columns

Sometimes it might be desirable to just throw a bunch of columns at a grid, letting them grow as necessary. It may also be desirable to have columns which only take up as much space as required for the content. To achieve this, we have two concepts, the first being auto columns, and the second being packed columns. Auto columns are specified with the .flexy-col-XX class, and packed columns with the .flexy-col-XX-packed class.
